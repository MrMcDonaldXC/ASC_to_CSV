# ASC to CSV 功能列表

## 一、项目概述

**项目名称**：ASC to CSV 转换与可视化工具
**版本**：v1.1.0
**功能定位**：CAN总线数据（ASC格式）处理、转换、可视化与分析

---

## 二、现有功能模块

### 1. 数据转换模块 (ConvertTab)

| 功能 | 描述 |
|------|------|
| ASC单文件模式 | 选择单个ASC文件进行转换 |
| ASC多文件拼接模式 | 添加多个ASC文件，按时间戳排序后拼接转换 |
| DBC文件管理 | 添加/删除DBC文件，用于CAN消息解析 |
| 采样间隔配置 | 设置数据采样间隔（秒） |
| 分组规则 | BatP+数字 → 独立分组；BatP+字母 → 独立分组；其他 → Others |
| CSV编码选择 | 支持 utf-8-sig, utf-8, gbk, gb2312 |
| 调试模式 | 启用后显示详细错误信息 |
| 配置保存/加载 | 将当前配置保存到config.json |

### 2. 数据可视化模块 (VisualizeTab)

| 功能 | 描述 |
|------|------|
| CSV文件选择 | 从下拉列表选择已转换的CSV文件 |
| 目录刷新 | 刷新CSV文件列表 |
| 数据列搜索 | 通过关键词过滤列名 |
| 图表缩放 | 0.1x ~ 5.0x 缩放控制 |
| 水平滚动 | 控制图表显示范围 |
| 十字参考线 | 显示鼠标位置的坐标值 |
| 鼠标坐标显示 | 实时显示X/Y坐标 |

### 3. 数据对比模块 (CompareTab)

| 功能 | 描述 |
|------|------|
| 多文件选择 | 选择多个CSV文件进行对比 |
| 列选择 | 选择要对比的数据列 |
| 图表同步 | 同步显示多个文件的对比图表 |

---

## 三、新增功能模块

### 4. 数据提取与导出模块 (ExportTab)

#### 4.1 列选择界面

| 功能 | 描述 |
|------|------|
| 文件选择 | 从已有CSV文件列表选择源文件 |
| 列名展示 | 清晰展示CSV文件中的所有列名 |
| 多选操作 | 支持Ctrl/Shift多选列 |
| 全选/取消全选 | 快捷操作按钮 |

#### 4.2 数据预览

| 功能 | 描述 |
|------|------|
| 预览窗口 | 弹窗显示所选列的数据样本（前10行） |
| 预览刷新 | 选择列变化时自动刷新预览 |

#### 4.3 导出配置

| 功能 | 描述 |
|------|------|
| 文件名设置 | 用户自定义导出文件名 |
| 保存路径 | 选择导出文件的保存目录 |
| 编码格式 | utf-8-sig, utf-8, gbk, gb2312 |

#### 4.4 导出执行

| 功能 | 描述 |
|------|------|
| 进度显示 | 显示导出进度 |
| 成功反馈 | 导出成功后显示提示信息 |
| 失败处理 | 错误信息显示与处理 |

## 四、技术架构

### 模块结构

```
asc_to_csv/
├── main_app.py          # 主程序入口
├── config.py            # 配置管理
├── core/
│   ├── csv_loader.py     # CSV数据加载
│   └── chart_manager.py  # 图表管理
├── ui/
│   ├── base.py           # 基础UI组件
│   ├── convert_tab.py    # 数据转换标签页
│   ├── visualize_tab.py  # 数据可视化标签页
│   ├── compare_tab.py    # 数据对比标签页
│   └── export_tab.py     # 数据导出标签页 (新增)
└── tests/               # 单元测试
```

### UI设计规范

- 框架：tkinter + ttk
- 布局：LabelFrame + Frame 嵌套
- 字体：Microsoft YaHei / SimHei
- 编码：UTF-8
- 配色：系统默认主题

---

## 五、数据流

```
ASC文件 → [ConvertTab] → CSV文件 → [VisualizeTab] → 图表
                              ↓
                         [ExportTab] → 部分列CSV文件
                              ↓
                         [CompareTab] → 对比图表
```
