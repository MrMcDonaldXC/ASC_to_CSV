│   │  ┌─────────────┐    ┌─────────────┐    ┌─────────────┐             │  │
│   │  │ ASC文件解析 │ →  │ DBC信号解码 │ →  │ CSV文件生成 │             │  │
│   │  └─────────────┘    └─────────────┘    └─────────────┘             │  │
│   └─────────────────────────────────────────────────────────────────────┘  │

### 2.2 主要特性

| 特性 | 说明 |
|------|------|
| **多DBC支持** | 支持同时加载多个DBC数据库文件 |
| **智能分组** | 自动按BatP模式对信号进行分组 |
| **空值填充** | 自动填充缺失数据，保证数据连续性 |
| **数据可视化** | 支持缩放、滚动、十字参考线等交互功能 |
| **多文件对比** | 支持多个CSV文件的数据对比分析 |
| **调试模式** | 提供详细错误信息，便于问题排查 |

|------|------|------|
| **Python** | 3.9+ | 仅源码运行需要 |

### 2.4 输入输出文件格式

#### 输入文件

| 文件类型 | 格式 | 说明 |
|----------|------|------|
| **ASC文件** | .asc | CAN总线日志文件 |
| **DBC文件** | .dbc | CAN数据库定义文件 |

| 文件类型 | 格式 | 说明 |

## 三、安装与配置指南

### 3.1 安装方式

本软件提供两种安装方式：

| 方式 | 适用场景 | 优点 |
|------|----------|------|
| **EXE版本** | 普通用户 | 无需安装Python，双击运行 |
| **源码版本** | 开发者 | 可自定义修改，便于调试 |

### 3.2 EXE版本安装

#### 步骤一：获取软件

```
1. 从发布页面下载 ASCtoCSV.exe 文件
2. 将文件保存到任意目录（如：D:\Tools\ASCtoCSV\）
```

#### 步骤二：准备DBC文件

```
1. 将DBC数据库文件复制到软件同目录或子目录
2. 建议目录结构：
   
   D:\Tools\ASCtoCSV\
   ├── ASCtoCSV.exe          ← 主程序
   ├── dbc\                  ← DBC文件目录
   │   ├── BMS.dbc
   │   └── VCU.dbc
   └── data\                 ← 数据文件目录
       ├── input.asc
       └── output\
```

#### 步骤三：验证安装

```
1. 双击 ASCtoCSV.exe
2. 如果程序正常启动并显示主界面，则安装成功
```

### 3.3 源码版本安装

#### 步骤一：安装Python

```
1. 下载 Python 3.9 或更高版本
   官网：https://www.python.org/downloads/

2. 安装时勾选 "Add Python to PATH"

3. 验证安装：
   打开命令提示符，输入：
   python --version
   应显示：Python 3.x.x
```

#### 步骤二：获取源码

```
1. 下载源码压缩包并解压
   或使用 git clone 命令
```

#### 步骤三：安装依赖

```bash
# 进入项目目录
cd d:\broserforcoding\asc_to_csv

# 安装依赖
pip install -r requirements.txt
```

#### 步骤四：运行程序

```bash
# 运行GUI版本
python main_app.py

# 或运行CLI版本
python main.py
```

### 3.4 配置文件说明

#### 3.4.1 配置文件位置

```
配置文件：config.json
位置：软件所在目录
```

#### 3.4.2 配置文件格式

```json
{
    "asc_file": "data/input.asc",
    "dbc_files": ["dbc/BMS.dbc", "dbc/VCU.dbc"],
    "output_dir": "output",
    "sample_interval": 0.1,
    "group_size": 5,
    "csv_encoding": "utf-8-sig",
    "debug": false
}
```

#### 3.4.3 配置参数说明

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `asc_file` | 字符串 | "" | ASC日志文件路径 |
| `dbc_files` | 数组 | [] | DBC文件路径列表 |
| `output_dir` | 字符串 | "" | CSV输出目录 |
| `sample_interval` | 数值 | 0.1 | 采样间隔（秒） |
| `group_size` | 整数 | 5 | 分组大小 |
| `csv_encoding` | 字符串 | "utf-8-sig" | CSV文件编码 |
| `debug` | 布尔 | false | 是否启用调试模式 |

### 3.5 验证安装成功

#### 功能验证清单

```
□ 1. 程序能够正常启动
□ 2. 界面显示正常，无乱码
□ 3. 能够选择ASC文件
□ 4. 能够添加DBC文件
□ 5. 能够执行转换操作
□ 6. 能够生成CSV文件
□ 7. 能够显示数据图表
```

#### 4.2.2 基本操作流程

**步骤一：选择ASC文件**

```
1. 点击 "浏览..." 按钮
2. 在文件选择对话框中选择ASC文件
3. 点击 "打开" 确认选择
```

**步骤二：添加DBC文件**

```
1. 点击 "添加" 按钮
2. 在文件选择对话框中选择DBC文件
3. 可重复添加多个DBC文件
4. 如需删除，选中列表中的文件后点击 "删除"
```

**步骤三：选择输出目录**

```
1. 点击 "浏览..." 按钮
2. 选择或创建输出目录
3. 点击 "确定" 确认
```

**步骤四：设置参数**

| 参数 | 设置建议 |
|------|----------|
| **采样间隔** | 0.1秒（默认），数据量大时可增大 |
| **编码** | utf-8-sig（推荐），兼容Excel |
| **调试模式** | 遇到问题时勾选 |

**步骤五：执行转换**

```
1. 点击 "开始转换" 按钮
2. 等待转换完成
3. 查看日志输出确认结果
```

#### 4.2.3 操作示例

```
1. ASC文件: 选择 "BMS_test_20240101.asc"
2. DBC文件: 添加 "BMS.dbc"
3. 输出目录: 选择 "D:\data\output\"
4. 采样间隔: 保持默认 0.1
5. 点击 "开始转换"
6. 等待完成，输出文件：

## 五、故障排除

### 5.1 常见问题列表

#### 问题1：程序无法启动

| 症状 | 可能原因 | 解决方法 |
|------|----------|----------|
| 双击无反应 | 杀毒软件拦截 | 添加到白名单 |
| 提示缺少DLL | 系统缺少运行库 | 安装VC++运行库 |
| 闪退 | 配置文件损坏 | 删除config.json重试 |

#### 问题2：转换失败

| 错误信息 | 可能原因 | 解决方法 |
|----------|----------|----------|
| "ASC文件不存在" | 文件路径错误 | 检查文件路径 |
| "DBC文件不存在" | DBC文件被移动/删除 | 重新添加DBC文件 |
| "无法识别文件编码" | 文件编码特殊 | 尝试转换文件编码 |
| "内存不足" | 文件过大 | 增大采样间隔 |

#### 问题3：图表无法显示

| 症状 | 可能原因 | 解决方法 |
|------|----------|----------|
| 图表空白 | CSV文件无数据 | 检查CSV文件内容 |
| 中文乱码 | 字体缺失 | 安装SimHei字体 |
| 图表卡顿 | 数据量过大 | 缩小查看范围 |

### 5.2 错误代码解释

| 错误代码 | 含义 | 处理建议 |
|----------|------|----------|
| E001 | ASC文件路径无效 | 检查文件是否存在 |
| E002 | DBC文件解析失败 | 检查DBC文件格式 |
| E003 | 输出目录无写入权限 | 更换输出目录 |
| E004 | 采样间隔无效 | 设置有效采样间隔 |
| E005 | 内存分配失败 | 减小数据量或增大采样间隔 |

   2. 分段处理大文件
   3. 确保内存充足（建议8GB以上）
```
